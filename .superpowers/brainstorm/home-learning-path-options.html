首页学习路径改版：3 个方向

目标：让初学者先知道“从哪里开始、为什么学这个、下一步去哪里”。

推荐 A

路径优先

顶部显示 3 条学习路径：入门、数论、泛型编程。
第一步线性查找
然后二分 / GCD
最后RSA / 矩阵幂
  • 最适合“我该先点哪个”的用户。
  • 不改变现有算法页面。
  • 首页会比现在更有导航含义。
方案 B

章节优先

保留书籍章节顺序，给每项增加难度、主题和前置知识。
3.2 筛法数论 · 初级
7.5 幂算法抽象 · 中级
13.4 RSA综合 · 高级
  • 最贴合图书目录。
  • 对读书用户友好。
  • 对零基础用户引导弱一些。
方案 C

主题仪表盘

首页按主题分区：查找、排序、数论、序列、图。
查找2 个

  • 扫描效率高。
  • 当前阶段会弱化书籍主线。
